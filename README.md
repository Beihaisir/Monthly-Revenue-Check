# 每月营收核对及自动汇总工具

Streamlit 应用，用于自动汇总餐饮门店多渠道支付数据，生成对账报表。

---

## 功能说明

1. **交互式上传**：支持上传主表及多个渠道账单（文件名可任意变化，只要表内字段结构一致即可）
2. **自动识别**：自动识别表头字段，支持 `.csv` / `.xls` / `.xlsx` 格式
3. **多Sheet检测**：当Excel文件包含多个工作表时，自动检测并弹窗让用户选择
4. **自动生成报表**：输出 `.xlsx` 格式，包含以下汇总列：
   - 当日金额合计
   - 各类支付累和
   - 当日易宝支付统计
   - 当日易宝到账(含手续费)
   - 当日美团外卖到账
   - 当日美团团购到账
   - 当日抖音到账
   - 当日淘宝闪购到账
   - 当日易宝支付到账差异
   - 当日美团外卖到账差异
   - 当日美团团购到账差异
   - 当日抖音团购到账差异
   - 当日淘宝闪购到账差异

---

## 各报表获取方式及数据说明

| 报表名称 | 获取路径 | 关键字段 | 数据说明 |
|---------|---------|---------|---------|
| **支付方式销售报表（主表）** | 门店管理后台 &gt; 统计报表 &gt; 销售日报-分支付 | 日期、支付方式名称、金额 | 门店每日各支付方式的收款汇总，作为对账主数据源 |
| **易宝账单** | 账户中心 &gt; 资金账单 &gt; 自助生成资金账单 &gt; 生成汇总资金账单 | 记账日期、业务类型、收入(元)、支出(元)、手续费(元) | 易宝支付渠道的收单、退款、手续费明细 |
| **美团外卖账单** | 美团外卖\|商家中心 &gt; 财务管理 &gt; 下载专区 &gt; 选择账单 | 账单日期、账单金额 | 美团外卖平台每日结算金额 |
| **美团团购账单** | 美团经营宝 &gt; 财务管理 &gt; 账单管理 &gt; 每日收益 &gt; 按日期汇总 | 日期、商家应得 | 美团团购平台每日商家应得金额 |
| **淘宝闪购账单** | 淘宝闪购商家版 &gt; 财务管理 &gt; 财务下载 &gt; 账单-汇总 | 账单日期、结算金额 | 淘宝闪购平台每日结算金额 |
| **抖音团购账单** | 抖音来客后台 &gt; 资金财务 &gt; 财务首页 &gt; 按日期 | 核销日期、商家应得 | 抖音团购平台每日核销后的商家应得金额 |
